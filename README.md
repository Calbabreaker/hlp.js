## Source files

All of the source files for hlp are located in src/

To compile the source files yourselves into build/hlp.js and build/hlp.min.js (minified version)

First install dependencies

```sh
npm install
```

Then run (uses webpack)

```sh
npm run build
```

For development

```sh
npm run dev
```
